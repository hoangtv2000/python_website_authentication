# Flask Web App
Web authentication and database tutorial.

## Setup & Installtion

Make sure you have the latest version of Python installed.

**And installed**:
+ flask
+ Flask-SQLAlchemy
+ flask-login

```bash
git clone <repo-url>
```

## Running The App

```bash
python main.py
```

## Viewing The App

Go to `http://127.0.0.1:5000`
